## Установка

1. Создайте виртуальное окружение:
```bash
py -3.12 -m venv venv
```

2. Активируйте виртуальное окружение:
```bash
# Windows PowerShell
venv\Scripts\Activate.ps1

# Windows CMD
venv\Scripts\activate.bat
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```
